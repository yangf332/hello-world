SiteMap
====================

### 类型
* XML for search engine bots
* HTML for user

## 格式
````
<?xml version="1.0" encoding="utf-8">
<urlset>
     <url>
          <loc>链接地址</loc>
          <lastmod>最后更新时间 2014-05-24T17:02:52+00:00</lastmod>
          <changefred>更新频率 Always|Hourly|Daily|Weekly|Monthly|Yearly</changefred>
          <priority>权重0.1-1.0</priority>
     </url>
</urlset>

````
